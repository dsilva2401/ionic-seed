Ionic Seed
==========


## Requirements
- npm
- gulp

## Installation
```bash
# npm dependencies
npm install
```

## Serve app
```bash
gulp serve
```

## Ionic commands
```bash
# Add platform
gulp ionic:add --platform [platform]

# Add plugin
gulp ionic:add --plugin [plugin]

# Run
gulp ionic:run --platform [platform]

# Build
gulp ionic:build --platform [platform]
```
